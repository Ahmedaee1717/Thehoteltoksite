Live AI Meeting Studio - Feature Tester

Use this page to test all features of the Live Meeting Studio.

Test 1: Create Demo Meeting

Creates a demo meeting with 10 transcript chunks, 3 speakers, sentiment analysis.

Test 2: Fetch Live Updates

Fetches transcripts, sentiment, and speaker stats for a meeting.

Test 3: AI Co-Pilot

Ask questions about the meeting transcript (requires OpenAI API key).

Test 4: Generate Sparkpage

Generate a structured summary document (requires OpenAI API key).

API Keys Required

AI Co-Pilot and Sparkpage features require OpenAI API key. Add it via: npx wrangler pages secret put OPENAI_API_KEY

How to Use

  1. Click "Create Demo Meeting" to generate a test meeting
  2. Copy the meeting ID from the result
  3. Use the meeting ID to test other features
  4. Open Live Meeting Studio to see real-time updates
